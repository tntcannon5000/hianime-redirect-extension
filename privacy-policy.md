# Privacy Policy

Last updated: 2026-04-02

## HiAnime Redirect Extension

This Chrome extension redirects `hianime.to` pages to the matching `aniwatchtv.to` page.

## Data collection

This extension does not collect, store, transmit, sell, or share personal data, browsing history, website content, user activity, authentication data, financial information, health information, or location data.

## Remote code and third parties

This extension does not use remote code, analytics, tracking scripts, or third-party advertising services.

## How the extension works

The extension uses Chrome's declarative request redirect rules to redirect supported `hianime.to` page requests to the corresponding `aniwatchtv.to` URL on the user's device.
