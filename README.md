# HiAnime to AniWatch Redirect

This is a minimal Chrome Manifest V3 extension that redirects:

- `https://hianime.to/something`
- `https://www.hianime.to/something`

to:

- `https://aniwatchtv.to/something`

## Load in Chrome

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this folder

## Package for Chrome Web Store

Zip the contents of this folder and upload the zip in the Chrome Web Store Developer Dashboard.
